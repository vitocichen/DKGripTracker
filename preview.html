📋 UI 元素说明

  • 图标明亮 — 还有充能可用（≥1层）
  • 图标变暗+去色 — 充能为 0，不可使用
  • 转圈遮罩(黑色扇形) — 冷却进度，顺时针旋转
  • 中央黄色数字 — 剩余冷却秒数（有充能时黄色）
  • 中央红色数字 — 剩余冷却秒数（0充能时红色）
  • 右下角白色数字 — 当前充能层数
  • 右下角红色数字 — 充能为 0 时变红
  • Shift+拖拽 — 移动图标位置（游戏内）
  • /dkgrip — 斜杠命令查看帮助
